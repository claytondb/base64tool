# Base64 Tool

A free online Base64 encoder and decoder for text, files, and images.

## Features

### Multiple Modes
- **Text** - Encode/decode plain text
- **File** - Convert any file to/from Base64
- **Image** - Handle images with preview
- **URL** - Encode/decode URL-safe Base64

### Actions
- Encode to Base64
- Decode from Base64
- Copy to clipboard
- Paste from clipboard
- Clear input

## Usage

Open `index.html` in any modern browser. No installation or build process required.

1. Select mode (Text, File, Image, URL)
2. Enter or paste content
3. Click Encode or Decode
4. Copy the result

## Use Cases

- Embedding images in CSS/HTML
- Encoding data for API requests
- Debugging Base64-encoded payloads
- Converting binary files for text transport

## Tech

- Pure HTML/CSS/JavaScript
- No external dependencies
- Works offline
- Responsive design
